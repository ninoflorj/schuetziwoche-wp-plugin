== Description ==

Anmeldungsplugin für die Schütziwoche der Pfadi Region Winterthur. 
Um das Plugin zu verwenden, benutze den Shortcode [schuetziwoche] auf einer Contentseite.
Einige Einstellungen müssen im Adminbereich gemacht werden.
